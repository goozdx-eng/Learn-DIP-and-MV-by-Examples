# 数字图像处理与机器视觉教程

从零构建数字图像处理知识体系，从课本概念到工程直觉。

---

## 这份教程解决什么问题

课本上的概念定义是精确的，但读完之后遇到实际问题还是反应不过来——知道"Canny边缘检测有四步"，但不知道"为什么需要这四步，每一步漏了会怎样"。这份教程的每个知识点都从真实困惑出发，推导设计原理，让概念变成可以推理的直觉。

## 教学设计原则

每章遵循 **困惑 → 类比 → 原理 → 验证 → 坑点** 五步结构：

- **困惑**：开篇提出一个真实会遇到的问题
- **类比**：用生活中的例子建立直觉
- **原理**：用数学或物理逻辑推导设计原因
- **验证**：每章结尾有检验题，做不对说明基础概念还没过关
- **坑点**：指出最容易走弯路的地方

> 如果读完一章没有"啊，原来是这样"的感受，说明这章还没读透。
